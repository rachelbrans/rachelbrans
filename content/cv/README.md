# CV Files

Place a PDF CV in this folder and point `cv.downloadHref` in `content/site-data.js` to it, for example:

```js
downloadHref: "./content/cv/rachel-brans-cv.pdf"
```

The current placeholder download is `rachel-branscombe-cv.pdf`, generated from `rachel-branscombe-cv.html`; replace the PDF when the final CV is ready.
