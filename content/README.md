# Portfolio Content

Edit `content/site-data.js` to update the portfolio without changing the layout code.

- Put replacement images in `content/images/`.
- Put a CV PDF in `content/cv/` and set `cv.downloadHref` in `site-data.js`.
- Add projects by copying an existing object in the `projects` array.
- Use `type: "image"`, `type: "video"`, `type: "article"`, or `type: "link"` to change how a project is labelled and opened.
- For video projects, set `embedUrl` to a YouTube, Vimeo, or other iframe embed URL.
